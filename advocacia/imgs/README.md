# Imagens da landing page – Advocacia

Coloque nesta pasta as imagens com os nomes abaixo:

| Arquivo      | Uso                          |
|-------------|------------------------------|
| `logo.svg`  | Logo do escritório (header)  |
| `hero.jpg`  | Imagem da primeira dobra     |
| `advogado.jpg` | Foto do advogado (seção Sobre) |

Recomendação: JPG ou WebP, resolução mínima 800px na menor dimensão. Se `logo.svg` não existir, o nome do escritório será exibido como texto.
